Project V4

What Changed?

For the final version of my project I mainly just cleaned some things up. I ended up not changing a whole lot to my site as I believe it has a cohesive format at the moment. I added some CSS to my hamburger label for mobile view. When you hover over the label the color will transition to the same background color of the card, indicating a call to action.

Resize the width of your browser to below 1000px to see the menu and test the hover action!

What was added?

I did not add any new pages for the final version of the site, but I added an iframe and a video element to my showcase page giving viewers a better perspective of some of the locations from the photos.

The style rules are in my CSS file under the images and media section note that I left.

Project V3

What Changed?

For version 3 of the site I decided to re-do my top navigation bar as a flex item. I added a media rule that collapses the links at 768px and adds a hamburger style drop-down menu for easier navigation on mobile. I stickied the nav bar to the top as well so the site is easier to navigate overall on all devices.

After reviewing feedback from last verison's discussion, I decided to clean-up my homepage a bit more and make things more uniform. Going forward I will begin to structure my 'about this verison' posts like this one.

What was added?

I added a new page titled 'Data'. On this new page I added a simple table with responsive properties to allow the data to be easily viewed. I documented all my style properties on my stlye sheet under the 'Style the table' note I added.

Other than the addition mentioned above, I only added some small things. I included some font awesome icons throughout and I was able to get a site logo up and working and yes it is a coffee emoji ‚òï.

Project V2

Overall, I had a good time messing around with different styling techniques for this portion of the project. I decided to make my new page a showcase of sorts, to put some of my own personal photos and to also help get a better grasp on grid style layouts with CSS.

I opted to continue using my card style layout after having trouble trying to redo everything via flexbox or grids. I really how easy it is to make your site more responsive using these techniques, so I would like to try and learn more and implement for the next version of the project.

After reviewing the feedback from Version 1, I also decided to do away with the about info being another column on the side. I like the one column, infinite scroll, style layout better. I feel like it doesn't try to unnecessarily pull your attention in one way or the other, so I appreciated that feedback!

Project V2

Overall, I had a good time messing around with different styling techniques for this portion of the project. I decided to make my new page a showcase of sorts, to put some of my own personal photos and to also help get a better grasp on grid style layouts with CSS. I opted to continue using my card style layout after having trouble trying to redo everything via flexbox or grids. I really how easy it is to make your site more responsive using these techniques, so I would like to try and learn more and implement for the next version of the project.

After reviewing the feedback from Version 1, I also decided to do away with the about info being another column on the side. I like the one column, infinite scroll, style layout better. I feel like it doesn't try to unnecessarily pull your attention in one way or the other, so I appreciated that feedback!
